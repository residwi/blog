# Personal Blog

A personal blog built with Jekyll and TailwindCSS, with a dedicated TIL (Today I Learned) section.

## Features

- Jekyll static site generator
- TailwindCSS for styling (configured in `tailwind.config.js`)
- Catppuccin Mocha color palette
- Client-side search powered by Lunr.js
- TIL collection as a separate Jekyll collection
- Tag pages generated by a small Jekyll plugin
- Code syntax highlighting via Rouge

## Prerequisites

- Ruby 3.x with Bundler
- Node.js with npm

## Local development

Install dependencies:

```bash
bundle install
npm install
```

Build the CSS, then run the site:

```bash
npm run dev
```

This builds the CSS once and starts `bundle exec jekyll serve --livereload`. Visit <http://localhost:4000>.

To watch the CSS in a separate terminal:

```bash
npm run watch:css
```

> Changes to `_config.yml` need a server restart. Jekyll does not reload it automatically.

## Creating content

Blog posts go in `_posts/YYYY-MM-DD-title.md` and TIL entries in `_til/YYYY-MM-DD-title.md`. The layout and permalink are applied automatically through `_config.yml` defaults.
